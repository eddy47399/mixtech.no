What is OpenVPN?

VPNs send data to and from the internet in virtual tunnels.
These digital tunnels both encapsulate your information, camouflaging it by wrapping it in layers of additional data, and encrypt it, translating it into a special code that only your intended destination can decipher. These two steps go a long way toward defending your data.

In addition to being the all-around safest VPN protocol, OpenVPN offers other advantages you can read about at www.openvpn.net

Why you should set up OpenVPN

I use OpenVPN for server and network administration. Proxmox servers comes with a web interface to controle each server. This web interface can be reached from my local lan network, or it can be configured to take a romote connection. The problem is that the web interface have no encryption.
With OpenVPN can i get into my lan network from anywhere and make changes without the risk of getting hacked.
